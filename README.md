#tic-tac-toe

Крестики-нолики — это классическая логическая игра для двух игроков на поле три на три клетки. Игроки по очереди ставят свои символы — крестики (X) и нолики (O) — в свободные клетки, стремясь выстроить три своих знака в ряд по горизонтали, вертикали или диагонали. Игра заканчивается победой одного из игроков при создании такой линии или ничьей, если все клетки поля заполнены, а выигрышной комбинации нет.
#Загрузка игры
Для загрузки игры необходимо скачать файлы main.cpp, tictactoe.cpp, tictactoe.h и в терминал ввести строку "g++ main.cpp tictactoe.cpp -o tictactoe.exe".
